Tato planeta mapuje vývoj mého programování i kariéry od prvních experimentů po týmovou profesionální praxi.

Cesta nebyla lineární. Začínal jsem u ručně psaného PHP, pokračoval přes JavaScript a Node.js automatizace a postupně se ustálil na moderním full-stack přístupu se SvelteKitem.

Spojující prvek všech období je stejný: stavět funkční nástroje, rychle iterovat a učit se z reálného provozu.
